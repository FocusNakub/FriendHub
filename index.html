🚀 Roadmap

แผนพัฒนาเกม Hide & Seek Camouflage อัปเดตตามสถานะของโปรเจกต์

Progress: v0.3.4 Alpha

ตอนนี้โปรเจกต์อยู่ในช่วงวางระบบหลัก: Lobby, Queue, Random Role, Camouflage และกำลังต่อยอด Catch System

✅ Lobby System ห้องรอผู้เล่น จุดเริ่มเกม และพื้นที่สำหรับ Queue
✅ Queue System ระบบเข้าคิว นับจำนวนผู้เล่น และเตรียมเริ่มรอบ
✅ Random Role สุ่ม Hunter และ Hider พร้อมแสดง Role UI
✅ Camouflage Base Hider สามารถแปลงเป็น Object เช่น Bush, Flower, Clover
🔄 Catch System กำลังทำระบบจับ ตรวจระยะ และสรุปผลการจับ
🔄 Lobby Room Polish ปรับห้อง Lobby ให้ดูดีขึ้น พร้อมป้าย Update และ Safe Zone
⏳ Ranking System ระบบคะแนน Win / Lose และ Leaderboard
⏳ Weekly Event อีเวนต์รายสัปดาห์สำหรับเล่นกับเพื่อน

Next Goals

1. ทำ Catch System ให้เล่นได้จริง ตรวจระยะ Hunter กับ Hider, ป้องกันจับผิดเป้าหมาย, อัปเดตสถานะผู้เล่น
2. ปรับ UI ให้ชัดขึ้น Role UI, Timer, Queue Count และ Transform Prompt
3. เพิ่มภาพประกอบ Key Art Lobby, Hunter, Hider, Gameplay, Catch System, UI และ Map
4. เตรียม v0.4.0 เป้าหมายคือให้เล่นทดสอบกับเพื่อนได้ครบ Loop: Lobby → Queue → Gameplay → End Round
