# Hide & Seek Camouflage Website

เปิดไฟล์ `index.html` ในโฟลเดอร์นี้ได้เลย

โครงสร้าง:
- index.html
- images/*.webp

วิธีเปลี่ยนภาพ:
1. เอาภาพใหม่ไปวางในโฟลเดอร์ images
2. ใช้ชื่อเดิม เช่น lobby.webp, gameplay.webp, hunter.webp
3. เปิด index.html ใหม่ เว็บจะอัปเดตภาพเอง

ข้อดี:
- ไม่ฝัง Base64 ใน HTML
- ภาพคมกว่า
- อัปเดตง่าย
- เหมาะกับทำเว็บโปรเจกต์ระยะยาว
